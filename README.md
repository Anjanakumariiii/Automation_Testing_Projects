# Automation-Testing Portfolio

This repository contains multiple automation projects demonstrating expertise in **web and API testing**, **dynamic web applications**, and automation best practices.  
The projects highlight skills in **Selenium, TestNG, Maven, API Testing, Page Object Model (POM), OOP, and POJO design**.

---

## Projects

### 1. Automation Projects for Web & API Testing
- **Tools & Technologies:** Selenium, TestNG, Maven, API Testing, POM, OOP  
- **Description:** Developed multiple automation mini-projects covering website testing, API validation, and dynamic element handling.  
  Implemented **Page Object Model (POM)** and **POJO classes** to ensure scalable, maintainable, and reusable automated test components.

---

### 2. Selenium Basic Automation
- **Tools & Technologies:** Selenium, TestNG, Maven, Java  
- **Description:** Automated basic web interactions including form filling, element validation, and dynamic content handling for multiple websites.

---

### 3. TestNG Suite Projects
- **Tools & Technologies:** Selenium, TestNG, Maven, Java  
- **Description:** Created advanced TestNG suites with data-driven testing, logging, and reporting to manage multiple test cases efficiently.

---

### 4. Dynamic Websites & POM Framework
- **Tools & Technologies:** Selenium, Maven, POM, OOP, Java  
- **Description:** Built frameworks for handling dynamic web elements and reusable page objects to maintain structured and scalable automation across complex web applications.

---

## Key Features
- Automation for websites and APIs using modern frameworks and tools  
- Modular and reusable test scripts following **POM architecture**  
- Robust handling of dynamic elements and complex workflows  
- Scalable, maintainable code using **OOP and design patterns**  
- Test reporting and logging using **TestNG**
